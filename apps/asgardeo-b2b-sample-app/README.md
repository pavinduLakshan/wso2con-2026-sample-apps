# VoyageOps

A Next.js starter for a B2B travel management platform secured with the Asgardeo Next.js SDK.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Copy the sample environment file and configure it with values from your Asgardeo application:

```bash
cp .env.example .env.local
```

3. Run the app:

```bash
npm run dev
```

4. Open `http://localhost:3000`.
