# WSO2CON 2026 – Sample Applications

This repository showcases the sample applications built for **WSO2CON 2026**, demonstrating different identity and access management scenarios.

## Available Applications

### 1. [asgardeo-b2b-sample-app](/apps/asgardeo-b2b-sample-app/)  

A B2B SaaS application that allows organizations to onboard their employees and manage travel allocation requests.

### 2. [asgardeo-b2c-sample-app](/apps/asgardeo-b2c-sample-app/)  

A consumer-facing application where users can explore hotel deals and booking offers, then complete reservations seamlessly.
